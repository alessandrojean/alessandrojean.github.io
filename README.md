Files to my personal site.

## Running local

To compile the assets and run Jekyll on local you
need to follow those steps:

- Install last version of [Node.js];
- Run above commands:

  ```bash
  # Install dependencies.
  $ yarn install
  # Run with hot reloading.
  $ yarn dev
  ```

[Node.js]: https://nodejs.org/
